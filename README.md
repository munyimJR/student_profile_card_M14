# student_profile_card_M14

A responsive Student Profile Card Flutter application built with responsive design, shimmer loading, custom dialog, and success notifications.

## Features
- **Responsive Sizing**: Optimized for multiple mobile screen resolutions using `flutter_screenutil`.
- **Shimmer Loading State**: Simulates a 2.5-second async data load with `shimmer` before rendering the profile card.
- **Interactive Profile Card**: Displays a customized avatar badge, student ID, name, and department.
- **Custom Dialog & SnackBar**: Includes "View Details" and "Mark Present" actions with polished UI notifications.

## Tech Stack
- Flutter & Dart
- `flutter_screenutil`
- `shimmer`
